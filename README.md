Apple Store E-commerce Web Application
This project is a digital Apple Store platform developed as a comprehensive e-commerce solution. It enables users to register, browse Apple items by category (Phones, Macs, Accessories, etc.), manage a shopping cart, and complete secure checkouts from the comfort of their homes.

User Authentication: Secure registration and login system with session management.



Dynamic Product Catalog: Browse products with detailed descriptions, pricing, and category filters (Mobile, MacBook, Watch, Air Pods).




Shopping Cart System: Add, remove, and update quantities of items with real-time total calculations.




Order Management: Full checkout process capturing delivery information and transaction history.


Responsive UI: A modern interface built with Bootstrap to ensure compatibility across various screen sizes.

üìä Database Architecture
The system is powered by a relational database consisting of four primary tables:


Person: Manages user credentials and registration details.


Product: Stores item details, stock levels, and category metadata.


Order: Tracks completed transactions and shipping information.


ShopCart: Links orders to specific products and quantities.
